#  Velocify Ride Sharing System

A secure ride-sharing web application built using Django and MySQL with strong authentication and admin analytics.

##  Tech Stack

* Frontend: HTML, CSS, Bootstrap
* Backend: Django (Python)
* Database: MySQL
* Security: HMAC + PBKDF2 password encryption
* Admin Dashboard: Driver + financial management

##  Features

* User & Driver login/signup
* Secure password hashing
* Ride booking system
* Admin dashboard with analytics
* Role-based authentication
* Financial tracking

##  How to run project

```bash
git clone https://github.com/evxn77/velocify.git
cd velocify
pip install -r requirements.txt
python manage.py runserver
```
